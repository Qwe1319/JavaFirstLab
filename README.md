Контейнер для целых чисел
## Класс IntContainer

### Основные методы:

- **`add(int value)`** - добавить элемент в контейнер
- **`get(int index)`** - получить элемент по индексу
- **`remove(int index)`** - удалить элемент и вернуть его значение
- **`size()`** - получить количество элементов
- **`isEmpty()`** - проверить, пуст ли контейнер
- **`clear()`** - удалить все элементы
- **`getCapacity()`** - получить текущую емкость
